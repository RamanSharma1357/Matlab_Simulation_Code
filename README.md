# Matlab Simulation Code

"[Federated Deep Neural Network for
Privacy-Preserving QBER Prediction in Quantum
Key Distribution Systems]"

1. Open MATLAB
2. Run the main file (e.g., main.m)
3. Results will be generated automatically
